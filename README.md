# 🇩🇪 Germany Affordability Index

##  Objective
To analyze the affordability of major German cities by comparing rent, salary, and inflation trends.

##  Key Questions
- Is Berlin still affordable compared to other German cities?
- How have rent and salaries evolved?
- Which cities offer the best affordability?

##  Data Sources
- Rent data (city-wise)
- Salary data
- Inflation data (Eurostat / other sources)

##  Methodology
1. Data Cleaning and Preprocessing
2. Merging datasets across cities and time
3. Exploratory Data Analysis (EDA)
4. Building an Affordability Index:
   Affordability = Salary / Rent
5. Visualization using Tableau / Power BI

##  Expected Outcomes
- Ranking of cities by affordability
- Insights into rent vs salary trends
- Data-driven conclusion on Berlin’s affordability
